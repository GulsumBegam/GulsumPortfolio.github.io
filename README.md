A fully responsive, single-file personal portfolio built with pure HTML5, CSS3, and vanilla JavaScript — no frameworks, no dependencies, no build tools.
Features a live matrix-style canvas animation, holographic avatar display with 3D tilt on hover, floating tech symbol particles with connection lines, typed role effect, scroll-triggered reveal animations, animated stat counters, magnetic skill tags, and a custom cursor with a lagging ring. Designed with a dark theme, CSS custom properties, and smooth micro-interactions throughout.
Tech: HTML5 · CSS3 · Canvas API · Vanilla JS · Google Fonts · IntersectionObserver · RequestAnimationFrame
